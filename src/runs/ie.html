Inclement Emerald Machamp Run

Inclement Emerald

Inclement Emerald Machamp Run is the chronology of a very special Pokémon Inclement Emerald Run. In this run, we'll only be using the Machop line of Pokémon to beat the game. That’s right! Only Machops, Machokes or Machamps can be used the entire length of Inclement Emerald.

Let's use only Machamps to beat Pokémon Inclement Emerald! What an idea!

Learn the Ruleset

The challenge

Complete the entire YDA-version of Pokémon: Inclement Emerald using 6 only Machamps!

Specific Rules

The ruleset is pretty simple, use only the Machop line of Pokémon to conquer the Inclement Emerald Rom hack. To help clarify what that really means, here’s a clear ruleset list:

  • More Caps
  • Normal Mode
  • The Starter must be a Machop
  • Randomized encounters (necessary to get our first 5 Machops)
  • Can only use Machop, Machoke or Machamp for battles.
  • Save states are allowed (even in the middle of battle)
