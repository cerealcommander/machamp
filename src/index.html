What is Machamp.CC?

Machamp.cc catalogs some very special Pokémon game runs. This site catalogs the Pokémon games that have been completed using only the Machop line. That’s right! Only Machops, Machokes or Machamps can be used the entire length of any Pokémon game by me, CerealCommander.

Common rules

To clarify exactly what we mean by "Machamp only" runs, here’s the official ruleset:

  • Can only use Machop, Machoke or Machamp the entire game.
  • Only Dittos and Machops can be caught (So like, you can't catch a Machoke, it’s illegal 😁)
  • No Skipping Fights
  • Do double battles if possible
  • All items can be used
  • Bag usage is allowed during battles
  • No moves are banned
  • Not Hardcore / No Perma Death
  • Docs are allowed

🥊 CLAUSE: Machamp.cc has a strong focus on Fighting-Type Mons. We use Fighting Mons when a Machop isn't available as starter.
